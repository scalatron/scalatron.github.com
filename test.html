Scalatron

Learn Scala with a Programming Game

Download this project as a .zip file Download this project as a tar.gz file

Scalatron is a multi-player programming game in which bot programs written in Scala compete against each other. It is an educational resource for individuals or teams that want to learn more about the Scala programming language or want to hone their Scala programming skills.

Follow @scalatron on Twitter.

April 30, 2012: Work in Progress

You Want To Learn Scala?

Install Scalatron and work through the step-by-step tutorial to create your first bot in Scala. Assumes no prior knowledge of the language.

Getting Started
Download the Scalatron game and unzip it. Go into the /bin directory and run Scalatron.jar. 		Scala Tutorial
A step-by-step tutorial teaches you Scala and helps you build your first bot.
The Botwar Game
Compete for energy units in a cell-based arena with edible and harmful plants and beasts. For details, see the Game Rules. 		How Scalatron Talks To Your Bot
Read the Protocol spec to understand exactly how the game and your bot interact.
Benchmark & High Scores
You can use the Scalatron Benchmark to rate your bot and compare it to others' in the high scores. 		Scalatron User Group
Visit the user group on Google Groups to discuss your bot strategies, post your code or publish your benchmark scores.

You Want To Make Scalatron Better?

Scalatron is a resource for the Scala community and is in the public domain. The source code is available here on github and contributions are welcome!

How To Build Scalatron
If you want to build Scalatron yourself from source code, check out the simple build instructions. 		How To Contribute
Fork Scalatron here on github and send pull requests. Bring your own ideas or check out the list of open issues, many of which contain pointers on how/where to get started.
The Scalatron API
The Scalatron game server exposes a simple RESTful web API that allows you to manage users, upload source code, build bots and track the leader board. Check out the API specification for details. 		Contact The Maintainer
Before embarking on something major, you can contact the maintainer and ask for feedback or tips on where to get started.
